# Mall Interactive Sales Deck

## Overview
An interactive browser-based sales deck for a global shopping destination. Designed to replace static presentations with an immersive, video-first experience.

## Features
- Cinematic hero section
- Interactive navigation
- Retail, Dining, Entertainment modules
- Event & partnership showcase
- Smooth animations using Framer Motion

## Tech Stack
- React + Vite
- Framer Motion
- React Scroll
